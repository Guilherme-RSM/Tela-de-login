# Tela-de-login
Essa é a primeira tela de Login que criei utilizando HTML e CSS para colocar em pratica o aprendizado<br> de uma vídeo Aula do Youtube no Canal do Elias Santana.
<Hr>
Esse pequeno projeto para prática de criação de tela de login, <br>consiste em um formulário simples no qual temos 1 input type text e 1 input type password.
